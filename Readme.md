# Amazon Homepage Clone

A frontend clone of the Amazon homepage built using **HTML5** and **CSS3**. This project was created to practice web page structure, layout design, and CSS styling by recreating the appearance of Amazon's homepage.

---

## Features

* Amazon-inspired homepage layout
* Navigation bar
* Search bar
* Hero banner
* Product sections
* Footer
* Clean and organized code
* Desktop layout

---

## Technologies Used

* HTML5
* CSS3

1. Clone the repository

```bash

```

2. Navigate to the project folder.

3. Open `index.html` in any modern web browser.
